# Phase 1 Scope

Phase 1 is a closed free MVP. It validates the core structure before paid sales.

## Included

- Public marketplace home
- Free material cards
- Material detail pages
- License badges
- Demo buyer library
- Entitlement-based library filtering
- Bundle pricing calculation
- Legal and seller-rule documents

## Excluded

- Real authentication
- File uploads
- Admin review workflow
- Paid checkout
- Seller payouts
- Private file storage
- Download watermarking

## Next Phase

Phase 2 should add invited seller accounts, draft material creation, file upload, and admin review before any paid sales are enabled.
