# Phase 1 Scope

Phase 1 is a free community MVP. It validates the material-sharing structure before paid sales.

## Included

- Public marketplace home
- Free community material feed
- Material detail pages
- License badges
- Demo buyer library
- Entitlement-based library filtering
- Bundle pricing calculation
- Legal and seller-rule documents
- Likes, comments, and download counts as static prototype signals

## Excluded

- Real authentication
- File uploads
- Admin review workflow
- Paid checkout
- Seller payouts
- Private file storage
- Download watermarking

## Next Phase

Phase 2 should add invited creator accounts, draft material creation, file upload, and admin review while keeping all public materials free. Paid sales remain a later phase after community demand is validated.
