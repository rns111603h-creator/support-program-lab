# Phase 2 Prototype Scope

Phase 2 in this branch is a free community-sharing visual prototype, not a production implementation.

## Included

- Seller dashboard mock
- Material creation mock
- Admin review queue mock
- Review queue filtering logic
- Seller submission status summary logic
- Navigation from the marketplace home
- Free material feed with likes, comments, and download counts
- Future paid-price notes kept as roadmap metadata only

## Intentionally Excluded

- Real authentication
- Database persistence
- File upload
- Form submission
- Approval/rejection mutations
- Payment onboarding
- Seller identity verification
- Paid checkout
- Paid download gating
- Seller payouts

## Purpose

This prototype shows the intended service flow:

1. A seller prepares a material draft.
2. The seller checks license and safety requirements.
3. The seller submits the material for review.
4. The admin reviews copyright, personal data, welfare-safety language, and license fit.
5. Approved materials become free community listings.
6. Paid single-item and bundle sales are evaluated later, after the free community has enough usage and creator demand.
